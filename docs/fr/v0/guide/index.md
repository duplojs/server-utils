# Introduction

La librairie `@duplojs/server-utils` fait partie de l’écosystème DuploJS. Elle fournit un ensemble d’utilitaires techniques réutilisables pour gérer des besoins courants côté serveur. L’objectif est de proposer des helpers simples, typés et cohérents pour limiter la duplication et faciliter la maintenance des projets.

## Rôle

`@duplojs/server-utils` sert de boîte à outils transverse dans l’écosystème DuploJS :

- centraliser des opérations système fiables (lecture/écriture, permissions, temporaires) ;
- exposer des API homogènes quel que soit l’environnement d’exécution ;
- fournir des types et des interfaces pour mieux structurer le code applicatif.
